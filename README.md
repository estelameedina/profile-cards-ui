# Profile Cards UI

Interfaz web desarrollada con HTML y CSS que muestra tarjetas de perfil de usuarios con enlaces a redes sociales.

---

## Tecnologías utilizadas
- HTML  
- CSS  

---

## Funcionalidades
- Diseño de tarjetas de perfil  
- Estructura visual limpia y organizada  
- Enlaces a redes sociales  
- Estilos personalizados para cada perfil  

---

## Objetivo
Practicar el diseño de interfaces web y la maquetación con HTML y CSS.
